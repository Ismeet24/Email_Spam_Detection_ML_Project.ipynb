# 📧 Email Spam Detection using Machine Learning

This project is developed as part of my internship at **Oasis Infobyte (OIBSIP Task 4)**.

## 🚀 Project Overview
This project aims to build a Machine Learning model that classifies emails as **Spam** or **Not Spam** using Natural Language Processing (NLP) techniques.

---

## 📂 Dataset
- The dataset contains labeled email messages:
  - **Spam (1)**
  - **Ham (0)**

---

## 🛠️ Technologies Used
- Python
- Google Colab
- Pandas, NumPy
- Scikit-learn
- Matplotlib & Seaborn

---

## ⚙️ Workflow
1. Data Cleaning & Preprocessing  
2. Text Vectorization using TF-IDF  
3. Model Training:
   - Naive Bayes
   - Logistic Regression  
4. Model Evaluation  
5. Real-time Spam Prediction  

---

## 📊 Model Performance
- Achieved high accuracy using Logistic Regression
- Compared multiple models for better results

---

## 💡 Features
✔ Clean and structured pipeline  
✔ Multiple model comparison  
✔ Visualization (Confusion Matrix & Graphs)  
✔ Real-time prediction system  
✔ Model saving for future use  

---

## ▶️ How to Run
1. Open the notebook in Google Colab  
2. Upload the dataset (`spam.csv`)  
3. Run all cells step-by-step  
4. Test with your own email text
